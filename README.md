# 🚀 Streamlit Data Science App Deployment (Render)

This project demonstrates how to build and deploy a **Python Data Science / AI application using Streamlit** on **Render (free cloud platform)**.
This is deployed at : https://document-q-a.onrender.com/

## 📌 Project Overview

This application provides an interactive **Streamlit UI** for running data science / AI workflows such as:

* 📊 Data analysis & visualization
* 🤖 Machine Learning / AI model interaction
* 🔍 (Optional) RAG-based document querying
* 🔑 API-based model usage (Hugging Face / OpenAI)

---

## 🛠️ Tech Stack

* **Frontend/UI**: Streamlit
* **Backend**: Python
* **Deployment**: Render
* **ML/AI Libraries**:

  * pandas
  * numpy
  * scikit-learn
  * huggingface_hub / openai
  * faiss (optional for vector search)
